# `tab_hover_preview_border_width = 1.0`

{{since('nightly')}}

The width, in pixels, of the border drawn around the vertical tab bar hover
preview card. See [show_tab_hover_preview](show_tab_hover_preview.md). The border
uses the pane split colour (`colors.split`).

The default is `1.0`.

```lua
config.tab_bar_placement = 'Left'
config.tab_hover_preview_border_width = 4
```
