# `tab_hover_preview_delay_ms = 350`

{{since('nightly')}}

The delay, in milliseconds, before the vertical tab bar hover preview card
appears once the pointer settles on a tab. See
[show_tab_hover_preview](show_tab_hover_preview.md).

The default is `350`.

```lua
config.tab_bar_placement = 'Left'
config.tab_hover_preview_delay_ms = 500
```
